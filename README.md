# 📊 CodeVision - Enterprise IT Operations Dashboard


A modern, responsive IT Operations Dashboard with secure authentication, real-time analytics, and interactive data visualizations. Monitor your projects, track KPIs, and manage workforce metrics all in one place.


## ✨ Features

### 🔐 Authentication System
- Secure login page with pre-filled demo credentials
- Session persistence using localStorage
- User-friendly logout functionality

### 📈 Key Performance Indicators (KPIs)
- **Active Projects** - Real-time project count
- **Billability Rate** - Resource utilization percentage
- **Attrition Rate** - Employee turnover tracking
- **Average Satisfaction Score** - Team happiness index

### 📊 Interactive Charts
- **Bar Chart** - Projects distribution by client country (USA, UK, India)
- **Line Chart** - Monthly billable hours trend analysis
- **Pie Chart** - Project type distribution (Fixed Price vs Time & Material)

### 🔍 Advanced Filtering
- **Department Filter** - Filter by UI/UX, Cloud, Software Dev, Support, QA, DevOps, Data Analytics
- **Client Country Filter** - Filter by USA, UK, India
- **Reset Filters** - One-click reset functionality

### ⚠️ Risk Management
- **At-Risk Projects Table** - Projects with overrun > 10%
- Color-coded risk badges for easy identification

### 📋 Additional Insights
- Top Performing Departments ranking
- Project Health Summary (On Track, At Risk, Delayed)
- Financial Summary with budget vs actual variance

### 🎨 UI/UX Highlights
- Glassmorphic design with backdrop blur
- Animated background with floating orbs
- Responsive layout for mobile & desktop
- Smooth hover effects and transitions
- Dark theme optimized for professional environments
